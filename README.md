# gokit-crud

## Objetive
CRUD practice in golang using the gokit hexagonal architecture toolkit.

## Run App
```bash docker-compose-init.sh```

## Stop App
```bash docker-compose-stop.sh```

## Run Test
### All Tests
```make test```

### Test ./app
```make test-app```

### Test ./database-app
```make test-database```

### Test ./token-app
```make test-token```
